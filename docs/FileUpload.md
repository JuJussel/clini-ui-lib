# File Upload

Example
```
<cui-file-upload accept="['.png']"></cui-file-upload>
```
Props:

- title: Title of File Upload Table
- accept: Accepted file types as array. default: ['.txt','.csv','.bmp','.png','.jpeg','.jpg','.pdf','.pptx','.xlsx','.docx']
